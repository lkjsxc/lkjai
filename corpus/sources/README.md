# Corpus Sources

## Purpose

Reviewed JSON source files live here. Training code expands these sources into
message rows with consistent provenance.

## Contents

- [public.json](public.json): approved public-import SFT seed entries.
- [public-pretrain.json](public-pretrain.json): active and reference public
  English pretraining sources.

## Rules

- Add source files only after documenting provenance and license.
- Prefer generated rows from docs and source code over hand-written static
  packs when the same behavior can be derived.
